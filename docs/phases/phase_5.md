# Phase 5: OrgDNA

## Goal
Design the target operating model aligned with restructuring recommendations.

## Prerequisites
- Phases 2, 3, and 4 gates passed

## Inputs
- Organizational data and headcount facts
- Outputs from Phases 2-4

## Process
1. Current state organizational mapping
2. Span of control analysis
3. Target operating model design
4. Transition planning

## Gate Criteria
- Current state documented
- Target model aligned with financial projections
- Transition risks assessed

## Outputs
- Current state org map
- Span of control analysis
- Target operating model
- Transition plan

## Artifacts
- `orgdna_v1`: Organizational design
