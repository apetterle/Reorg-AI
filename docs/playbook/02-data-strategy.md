# Chapter 02: Data Strategy

## Objectives
- Define data requirements for the engagement
- Plan data collection approach
- Establish data quality standards

## Data Categories

| Category | Examples | Phases |
|---|---|---|
| Financial | P&L, balance sheet, cost reports | 0, 1, 2 |
| Headcount | Org charts, HR reports, payroll | 0, 1, 5 |
| Technology | License inventory, IT spend, contracts | 3 |
| Operational | Process maps, KPI dashboards | 1, 2 |
| Contracts | Vendor agreements, SLAs | 2, 3 |

## Quality Standards
- Data should be recent (within last 12 months)
- Financial data should be reconciled
- Headcount data should match payroll records
- All PII must be identified and handled per policy
