# Chapter 04: Baseline Establishment

## Objectives
- Extract structured facts from clean documents
- Establish baseline KPIs for the engagement
- Curate facts through analyst review

## Fact Extraction
- Automated extraction from CSV/XLSX (per-column facts)
- Text-based extraction from documents (LLM-assisted)
- Each fact links to source evidence

## Fact Curation
- Review proposed facts for accuracy
- Approve, reject, or edit extracted facts
- Ensure confidence scores are appropriate

## Baseline KPIs
- Run `baseline_kpis` numeric block on approved facts
- Normalize units and periods for comparability
- Identify gaps in KPI coverage
