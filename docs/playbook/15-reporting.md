# Chapter 15: Reporting

## Objectives
- Manage artifacts across phases
- Generate comprehensive reports
- Export deliverables in required formats

## Artifact Management
- Artifacts are grouped by phase and versioned
- Each version is immutable; new runs create new versions
- Evidence links maintain traceability

## Report Generation
1. Select phases and artifacts to include
2. Choose template (executive or technical)
3. Preview and review compiled report
4. Lock version and export (JSON/Markdown/HTML)
