# Chapter 16: Data Privacy

## Objectives
- Ensure LGPD compliance throughout engagement
- Handle PII according to fail-closed policy
- Maintain audit trail for compliance

## Key Procedures
- All uploads automatically scanned for PII
- Blocked documents cannot be used for extraction
- Sanitization removes PII before processing
- Evidence snippets always stored with redaction
- Retention policies enforce data lifecycle management

See `docs/policies/PII_LGPD.md` and `docs/policies/DATA_RETENTION.md` for full policies.
