# Chapter 17: Team Collaboration

## Objectives
- Manage team access and roles
- Enable efficient collaboration
- Maintain security boundaries

## Role-Based Access
| Role | Capabilities |
|---|---|
| Owner | Full access, billing, tenant management |
| Admin | Member management, project configuration |
| Analyst | Data upload, fact curation, phase execution |
| Viewer | Read-only access to outputs and reports |

## Workspace Management
- Invite members via email/link
- Manage roles through workspace settings
- Track activity through audit log
