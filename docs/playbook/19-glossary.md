# Chapter 19: Glossary

| Term | Definition |
|---|---|
| **Artifact** | A versioned output generated by a phase (e.g., ValueScope summary, ZeroBase analysis) |
| **Confidence** | A 0.0-1.0 score indicating reliability of an extracted fact |
| **Evidence** | A reference linking a fact to its source document location |
| **Fact** | An atomic, verifiable data point extracted from a document |
| **Gate** | A quality checkpoint that must pass before proceeding to the next phase |
| **KPI** | Key Performance Indicator; a quantifiable metric |
| **LGPD** | Lei Geral de Protecao de Dados (Brazilian data protection law) |
| **Lineage** | The traceability chain from source document through fact to artifact |
| **Numeric Block** | A deterministic computation module producing reproducible outputs |
| **Phase** | One of 8 stages (0-7) in the restructuring methodology |
| **PII** | Personally Identifiable Information |
| **RBAC** | Role-Based Access Control |
| **Sanitization** | The process of removing PII from documents |
| **Snippet** | A text excerpt from a document used as evidence |
| **Tenant** | An isolated workspace for a client engagement |
| **ValueScope** | Phase 1; initial value assessment and opportunity sizing |
| **ZeroBase** | Phase 2; zero-based cost analysis methodology |
