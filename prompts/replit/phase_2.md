# Phase 2: ZeroBase Rebuild — Implementation Prompt

## Objective
Perform zero-based cost analysis by rebuilding cost structures from scratch, identifying redundancies and optimization opportunities.

## Prerequisites
- Phase 1 (ValueScope) completed and gate passed

## Required Inputs
- Approved facts from Phase 1 (cost-related KPIs)
- Cost center breakdowns (from classified documents)
- Headcount reports

## Implementation Steps

1. **Cost Categorization**: Group approved cost facts by category (personnel, technology, facilities, services, etc.).

2. **Zero-Base Analysis**: For each cost category, assess necessity from zero — justify each cost item against business objectives.

3. **Redundancy Detection**: Identify duplicate functions, overlapping services, and consolidation opportunities.

4. **Savings Estimation**: Compute potential savings per category with confidence-weighted estimates.

5. **Artifact Generation**: Store `zerobase_v1` artifact with categorized analysis and savings projections.

## Gate Criteria
- All major cost categories analyzed
- Savings estimates have evidence backing
- Analyst review of zero-base recommendations

## Outputs
- Cost category breakdown
- Zero-base justification matrix
- Savings opportunity register
- ZeroBase rebuild artifact
