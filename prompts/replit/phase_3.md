# Phase 3: SmartStack — Implementation Prompt

## Objective
Optimize the technology stack by analyzing current technology spend, identifying consolidation opportunities, and recommending a rationalized stack.

## Prerequisites
- Phase 2 (ZeroBase Rebuild) completed and gate passed

## Required Inputs
- Technology-related facts from previous phases
- Software/license inventory data
- IT infrastructure cost data

## Implementation Steps

1. **Technology Inventory**: Catalog all identified technology assets, licenses, and services from extracted facts.

2. **Overlap Analysis**: Identify functional overlaps between tools and systems.

3. **Consolidation Opportunities**: Recommend tool consolidations with estimated savings.

4. **Migration Risk Assessment**: Assess risks and effort for each consolidation recommendation.

5. **Artifact Generation**: Store `smartstack_v1` artifact with technology rationalization plan.

## Gate Criteria
- Technology inventory complete
- Consolidation opportunities sized with evidence
- Risk assessment for each recommendation
- Analyst sign-off

## Outputs
- Technology inventory
- Overlap analysis matrix
- Consolidation recommendations with ROI
- SmartStack artifact
