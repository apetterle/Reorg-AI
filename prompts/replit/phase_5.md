# Phase 5: OrgDNA — Implementation Prompt

## Objective
Analyze the organizational structure and design a target operating model aligned with the restructuring recommendations from Phases 2–4.

## Prerequisites
- Phases 2, 3, and 4 completed

## Required Inputs
- Headcount and organizational data from extracted facts
- Role/function inventories
- Outputs from ZeroBase, SmartStack, and ValueCase phases

## Implementation Steps

1. **Current State Mapping**: Map current organizational structure from available data (headcount reports, org charts).

2. **Span of Control Analysis**: Analyze management layers, spans of control, and reporting relationships.

3. **Target Operating Model**: Design future-state organization aligned with restructuring recommendations.

4. **Transition Planning**: Identify role changes, redeployments, and capability gaps.

5. **Artifact Generation**: Store `orgdna_v1` artifact with current/target state comparison and transition plan.

## Gate Criteria
- Current state org structure documented
- Target model aligned with financial projections from ValueCase
- Transition risks assessed
- HR/legal compliance considerations addressed

## Outputs
- Current state org map
- Span of control analysis
- Target operating model
- Transition plan
- OrgDNA artifact
